# 🌐 Portafolio Web Personal

Portafolio web básico desarrollado con **HTML y CSS**, diseñado para presentar de forma clara, visual y estructurada mis proyectos, habilidades y perfil profesional.

## 🚀 Descripción

Este proyecto surge como una solución temporal optimizada para integrarse en mi CV, permitiendo mostrar de manera efectiva mi perfil como desarrollador mientras evoluciona hacia una versión más robusta, dinámica y escalable.

## 🎯 Objetivos

- Centralizar la presentación de proyectos personales
- Exponer habilidades técnicas de forma visual
- Fortalecer la presencia profesional digital
- Servir como base para futuras mejoras (JavaScript, frameworks, backend)

## 🛠️ Tecnologías Utilizadas

- HTML5  
- CSS3  

## 📌 Características

- Diseño limpio y enfocado en la experiencia visual  
- Estructura clara y fácil de navegar  
- Adaptado como recurso estratégico para CV  
- Código simple, mantenible y escalable  

## 🔮 Evolución del Proyecto

Este portafolio está concebido como una base inicial. A futuro se proyecta integrar:

- Interactividad con JavaScript  
- Consumo de APIs  
- Animaciones avanzadas  
- Migración a frameworks modernos (React, Next.js)  
- Optimización SEO y rendimiento  

## 📄 Uso

Este proyecto puede ser utilizado como plantilla base para portafolios personales, especialmente en etapas iniciales de desarrollo profesional.
